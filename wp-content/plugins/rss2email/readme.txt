  - ИНФО ---------------------------------------------------
  
  • Название плагина : rss2email.ru
  • Адрес: http://maxsite.org/
  • Автор: MaxSite.org
  • Версия: 1.1
  • Совместимость: все версии WordPress

  Данный плагин предназначен для размещения на страницах 
  вашего блога формы подписки, кнопки или счетчика сервиса 
  rss2email.ru. 
  
  Файлы плагина сохранены в кодировке UTF-8. Если ваш блог
  имеет другую, то вам нужно пересохранить файлы в нужной 
  вам кодировке.
  
  
  - УСТАНОВКА ----------------------------------------------

  1. Распакуйте архив (с каталогом "rss2email") в каталог 
     плагинов WordPress (/wp-content/plugins).
  2. Зайдите в админ-панель и в закладке "Плагины" 
     активируйте плагин "rss2email.ru".
  3. На закладке "Настройки" -> "rss2email.ru" вы можете 
     выбрать понравившуюся картинку, счетчик и форму 
     подписки.
  4. К особенностям данного плагина относится тот момент, 
     что для размещения счетчика, нужно зарегистрировать 
     вашу rss-ленту в сервисе rss2email.ru. Для этого вам 
     достаточно нажать кнопку "Получить номер ленты". Если 
     поле "Адрес RSS" не заполнено, то будет осуществлена 
     проверка rss-ленты вашего блога по-умолчанию. Если же 
     вы хотите разместить форму или счетчик для другого 
     адреса, например зарегистрированного в feedburner.com, 
     то введите его в указанное поле и получите для него
     номер.
     Если вы не получите номер, то счетчик и форма подписки 
     могут работать некорректно.
  
  
  - ИСПОЛЬЗОВАНИЕ ------------------------------------------
  
  1. Если ваш WordPress-шаблон поддерживает виджеты, то вы 
     можете разместить форму или кнопку (счетчик) с помощью 
     виджета.
  2. Для ручного размещения кнопки или счетчика используйте 
     следующий код:
  
  Форма подписки:
  <?php
  if ( function_exists('rss2email_form') ) { 
      rss2email_form();
  }
  ?>
  
  Кнопка/счетчик:
  <?php
    if ( function_exists('rss2email_button') ) { 
       rss2email_button();
    }
  ?>

  
  - УДАЛЕНИЕ -----------------------------------------------
  
  1. Деактивируйте плагин в админ-панели. При этом будут 
     удалены все настройки этого плагина.
  2. Удалите файлы плагина с сервера.
